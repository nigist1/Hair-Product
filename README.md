# Hair Product E‑commerce (prod)

A minimal full‑stack e‑commerce for hair products. Backend is an Express + MongoDB API; frontend is a React app built with Vite and Tailwind.

Features

- User authentication (register/login)
- Product listing and detail pages
- Cart, checkout and order management

Tech stack

- Backend: Node.js, Express, Mongoose, JWT
- Frontend: React, Vite, Tailwind CSS
